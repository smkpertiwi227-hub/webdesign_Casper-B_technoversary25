### 5.ğŸ’¬ Smart Chatbot Wisata Kuningan
AlamNusa dilengkapi fitur **chatbot berbasis JavaScript murni tanpa backend**, dirancang untuk membantu pengunjung mendapatkan informasi cepat mengenai destinasi wisata di Kuningan.
Chatbot bekerja menggunakan **metode rule-based matching**, yaitu mencocokkan kata kunci dalam input pengguna dengan daftar jawaban yang sudah ditentukan. Sistem ini ringan, cepat, dan dapat berjalan sepenuhnya di browser tanpa internet tambahan atau server eksternal.

#### Cara Kerja Teknis Singkat
- Pengguna mengetik pertanyaan ke input chatbot
- JavaScript membaca teks dan mencari **keywords**
- Jika keyword ditemukan, chatbot memberikan jawaban sesuai data yang tersedia
- Jika tidak cocok, chatbot menampilkan pesan fallback (â€œMaaf, saya belum punya info ituâ€)

#### Contoh pertanyaan yang dapat dicoba

| Kategori | Contoh Input | Respon yang Diharapkan |
|---------|--------------|------------------------|
| Info tempat wisata | "Apa itu Telaga Biru Cicerem?" | Penjelasan singkat + karakteristik tempat |
| Lokasi wisata | "Dimana letak Curug Putri?" | Lokasi + kecamatan / petunjuk arah dasar |
| Rekomendasi | "Rekomendasi tempat buat healing dong" | Daftar destinasi bernuansa santai |
| Aktivitas | "Tempat buat tracking?" | Rekomendasi gunung atau adventure spot |
| Jam buka/opsi wisata | "Cibulan buka jam berapa?" | Jam operasional standar |
| Tiket & harga | "Berapa tiket masuk Cicerem?" | Estimasi harga tiket |
| Area populer | "Wisata paling ramai di Kuningan?" | Suggestion hot-spot |
| Bantuan | "Cara pakai web ini gimana?" | Penjelasan fitur website |
| Fallback | Pertanyaan diluar data chatbot | Jawaban default: chatbot belum memiliki info |
